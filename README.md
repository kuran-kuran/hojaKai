# Z80逆アセンブラ hoja 改

## はじめに
これはをZ80逆アセンブラ hojaをkuran_kuranが改良(?)したものです。  
私が手を入れた事によって予期せぬバグも一緒に入った可能性があります。  
hojaは非常に高機能なZ80逆アセンブラですがこれで逆アセンブルしたプログラムリストは私が使っているz80asでエラーになりました。  
そこでエラーにならないように何個かオプションを追加しました。  

## オプション追加
以下のオプションを追加しました。  
### -p
文字列を'ではなく"で囲む  

### -q
10進数にDを付けない  

## z80asでアセンブルできるようにするためのオプション
hoja -oXXXX -d -g -u -p -q input.bin > output.asm  
-o の XXXXは開始アドレス
